# Website Aster - Frontend
O website da Aster é uma plataforma institucional criada para apresentar os produtos, planos, pacotes e funcionalidades dos softwares de criação, reunindo informações sobre cada ferramenta e planos de aquisição para a apresentação do ecossistema Aster.

## Instalação
#### Instale o Node (caso ainda não tenha):
Confira: <https://www.nodejs.org>


#### Instale as dependências:  
``npm install``


#### Para rodar a aplicação execute:  
``npm run dev``

> Acesse em ``http://localhost:5000``
